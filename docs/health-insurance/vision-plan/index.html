    眼科保險簡介 #

    最後更新日期2022.12.20

    眼鏡、隱形眼鏡可在台灣多買幾份備份替代此保險。眼睛檢查相當完整，則是在台灣須要自費項目。通常建議兩年一次，因此當年度有需求可以投保。

    保險項目 #

    眼科保險主要用於報銷/折抵以下費用：

    • 驗光：配眼鏡、隱形眼鏡強制，驗光的結果可以兩年內重複使用。
    • 眼睛檢查：包含散瞳、眼底鏡、視覺功能和眼睛健康測試，包括深度知覺、色覺、周邊。以上檢測在台灣是自費項目。
    • 眼鏡（配鏡架和鏡片）、隱形眼鏡：依照不同等級有不同的給付比例。

    眼睛疾病診斷治療
    眼睛疾病診斷治療是使用醫療保險而非眼科保險。

    視網膜成像
    如果不想要點散瞳做眼睛檢查，視網膜成像（Retinal imaging）會額外收費。

    校外眼科保險 #

    如果臨時有配眼鏡需求但沒有學校保險的情況下可考慮購買校外眼科保險，價格和福利沒有比學校學生保險差太多，甚至有部分權益更優秀。惟須注意，購買校外保險前需要確定是否有Waiting Period。

    案例介紹 #

    以UnitedHealthcare Vision - Plan B為例，沒有Waiting Period，保滿一年後可以任意終止。總體上沒有比學校學生保險差太多，臨時有需求仍可投保。

    • 保費： $15.7/月。
    • 眼睛檢查$10。
    • 同個年度可同時報銷眼鏡與隱形眼鏡。
    • 隱形眼鏡最多報$150，Costco一盒90片日拋約$70多。
    • 鏡框報銷$150，Costco一副約$120。
    • 鏡片只報銷最基礎款，Copayment為$10，其於鍍膜、較薄的鏡片（High index）有不同的Copayment。
    UnitedHealthcare Vision Plan

    • 保費： $7.54/月。
    • 眼睛檢查免費。
    • 同個年度可同時報銷眼鏡與隱形眼鏡。
    • 隱形眼鏡最多報$100，Costco一盒90片日拋約$70多。
    • 鏡框：Davis Vision’s Collection的Fashion、Designer的Copymant為$0、$20、$40。其餘至多報銷$100。
    • 鏡片報銷最基礎款Copayment $0，其於鍍膜、較薄的鏡片（High index）有不同的Copayment。
