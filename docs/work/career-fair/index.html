    校園徵才 #

    林青億主協編
    最後更新日期2021.06.04

    疫情關係，徵才活動可能以Virtual為主。

    Pitt #

    詳情參閱Pitt網站。

    Career Fair #

    Career Fair分別在Fall跟Spring有兩場大型徵才活動，主要分為3個領域，日期分開：A (Arts, Sciences, Business)、E (Engineering)、C (Computing & Information)。

    • 2021 Fall： TBD，通常在九月底。
    • 2022 Spring: TBD，通常在二月中。

    Student Employment Fair #

    Student Employment Fair主要提供校園實習、校園打工、當地工作機會、以及聯邦研究職位。舉辦於八月底。

    Greater Pittsburgh Law Fair #

    舉辦於十月中。

    CMU #

    CMU徵才僅限CMU學生及校友參加，詳情參閱CMU網站或Hankdshake。

    Encompass #

    Encompass是混合型徵才活動，每年Fall跟Spring各有一次。

    • 2021 Spring: 1/26-1/27。
    • 2021 Fall: 9/13-14。

    Technical Opportunities Conference #

    Technical Opportunities Conference (TOC)主要是提供技術相關的徵才活動，通常舉辦於Fall。

    • 2021 Fall: TBD。

    Tepper Meetup #

    Tepper Meetup主要是提供商學院或經濟主修相關的徵才活動，通常舉辦於Fall。

    • 2021 Fall: 9/9。
