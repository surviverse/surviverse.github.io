    房型介紹 #

    最後更新日期2021.05.24

    租屋常見的名詞：

    • Studio/Efficiency：廚房與臥房相通、沒有間隔，煮飯後全房內會有油煙味。
    • 1b1b：One-bedroom and one-bathroom、一房一廳，廚房與臥房分開（0.5b通常指的是沒浴室的廁所）。
    • Apartment：物業管理所擁有的公寓，屬於出租導向公寓。
    • Condominium：簡稱Condo，私人公寓但也可能有物業管理維護公共區域。
    • House：獨立房屋，通常有好幾間房間、車庫、草坪和院子。
    • Town House：類似於House，但數間連排共用小庭院，通常沒有獨立院子。

    Apartment與House比較 #

    ApartmentHouse
    所有權
    物業管理公司統一管理。私人擁有，有時為物業仲介管理。
    空間
    欠缺儲物空間。經常因客廳出租而更加缺乏活動空間。通常有地下室作為儲物空間、車庫、草坪和院子。
    管理
    通常可以24hr提交維修，維修迅速。通常需要跟屋主商討、安排維修事宜。
    找房與入住
    通常有精美的官網可以導覽，也通常有學長姐傳承家具。惟須注意，長時間承接租約，可能因欠缺務業整理而屋況不佳。每間房子差異性極大且通常承接學長姐機會較少，需要花費心思尋找以及重新購買家具。通常入住前，房東都會請人整理房子。
    環境
    可有找左鄰右舍聚會，但同時也可能會受到隔壁或樓上的噪音干擾。主要與室友共同生活，通常較為安靜。
    房租
    除住客廳（$500~$600）外，通常每月租金約在$700~$1,300/人。通常每月租金約在$550~$850/人。
    簡評
    通常找屋、入住方便以及擁有物業管理，但通常價格較高。House通常有較大的空間與價格優勢，但環境整潔以及租屋糾紛較為頻繁。
